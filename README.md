<a href="https://raw.githubusercontent.com/karsten314159/monster-scan/master/target/read-1.0-SNAPSHOT-jar-with-dependencies.jar">Download the JAR file</a>

<br />

Supported Languages:

<ul>
  <li>English</li>
  <li>German</li>
</ul

<pre>java -jar "path/to/jar-file/monster-scan.jar"</pre>
